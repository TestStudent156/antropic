# Anthropic Copilot Backup

Personal backup repository for Copilot session data, configurations, and notes.

## Structure

```
├── sessions/      # Exported session summaries and checkpoints
├── configs/       # Configuration backups (docker-compose, .env templates, etc.)
├── notes/         # Technical notes and documentation
└── .github/       # GitHub configs including copilot-instructions.md
```

## Usage

This repository stores important configurations and session knowledge for future reference.
